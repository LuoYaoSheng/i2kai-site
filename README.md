# i2kai-site

i2kai.com 个人官网，基于 Astro 5 构建。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

推送到 GitHub 后，GitHub Actions 自动构建并部署到 GitHub Pages。

## 内容管理

- 博客文章：`src/content/blog/` — Markdown 文件
- 视频笔记：`src/content/videos/` — Markdown 文件 + iframe 嵌入

## 技术栈

- [Astro 5](https://astro.build/)
- 纯 CSS（暖色系设计语言）
- Content Collections
- GitHub Pages + GitHub Actions
